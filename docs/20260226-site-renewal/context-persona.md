# ターゲットペルソナ定義

## プライマリターゲット: 買い手（大学・研究機関の研究者/技術者）

### ペルソナ1: 大学の准教授（40代）

- **所属**: 国立大学 理学研究科
- **予算**: 科研費（基盤C: 300-500万円/年）
- **課題**: 新品のHPLCは300万円超えで予算の大半を占める。学生の実験用に台数を増やしたいが新品では無理
- **情報収集**: Google検索 → 比較検討 → 問い合わせ
- **決め手**: 価格（予算内）、品質（動作確認済み）、納期（年度内）
- **不安**: 中古の品質は大丈夫か、故障したらどうするか

### ペルソナ2: 製薬企業の品質管理担当（30代）

- **所属**: 中堅製薬メーカー 品質管理部
- **予算**: 部門予算（設備投資枠）
- **課題**: 新規ラインの品質試験設備を短納期で立ち上げる必要。新品だと納期3ヶ月以上
- **情報収集**: 業界の知人 → Web検索 → 複数社見積
- **決め手**: 納期（2週間以内）、品質文書の有無、保証
- **不安**: 中古で薬事対応できるか、校正証明書は出るか

### ペルソナ3: 公的研究機関の研究員（50代）

- **所属**: 地方公設試験研究機関
- **予算**: 自治体予算（年度末消化あり）
- **課題**: 年度末に余った予算で来期の分析能力を強化したい。購入手続きに時間がかかる
- **情報収集**: 過去の取引先 → Web検索
- **決め手**: 年度内納品、見積書の迅速発行、在庫があること
- **不安**: 入札手続きとの整合、予算区分

### ペルソナ4: バイオスタートアップのCTO（30代）

- **所属**: 創業2年のバイオテック
- **予算**: 資金調達後のラボ立ち上げ資金（限られている）
- **課題**: 限られた資金でラボを立ち上げ、分析環境を構築する必要がある
- **情報収集**: スタートアップコミュニティ → Web検索
- **決め手**: コスパ（投資効率）、まとめて揃えられるか
- **不安**: 中古でちゃんと動くか、メンテは自分たちでできるか

## 共通する行動パターン

1. まずGoogle検索で「HPLC 中古」「遠心分離機 中古」等で検索
2. いくつかのサイトを比較
3. 在庫・価格・状態を確認
4. 検討リスト（お気に入り）に追加
5. 問い合わせ（見積依頼）
6. 社内稟議/購入手続き

## 共通する重視ポイント

- **価格の透明性**: いくらかすぐわかる、または問い合わせが簡単
- **品質への安心感**: 状態ランク、写真の充実、保証の有無
- **在庫の充実度**: 探している機器がありそうか
- **問い合わせの手軽さ**: フォームが簡単、電話番号がわかる
- **納期の速さ**: 在庫がある＝すぐ届く
