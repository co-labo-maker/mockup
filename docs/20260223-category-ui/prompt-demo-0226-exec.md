全中カテゴリに小カテゴリが必要なわけではない。商品数が多く、さらに細分化が意味のある中カテゴリにのみ追加すること。
目安: 主要な中カテゴリに2〜5個程度の小カテゴリ。

# Phase 3: HTMLプロトタイプ生成（4画面）

Phase 1 の調査結果を踏まえて、以下4画面のプロトタイプを生成せよ。

## デザイン方針

- **ベース**: モノタロウのテキストベース・シンプル・機能的なUI
- **差別化**: 中古機器なので写真を大きく表示。状態（A/B/C）やスペックが見やすいように
- **パーツデザイン**: ZOZOの字・余白詰め気味のトンマナを参考に
- **色・ボタン**: 現サイト（Phase 1-1の調査結果）を踏襲。ただし厳密でなくてOK（Figmaで最終調整する）
- **PC版のみ**: 幅1280px想定。レスポンシブ不要

## 画面1: prototype-category-pulldown.html

**検索ボックス横のカテゴリプルダウン**

要件:

- ヘッダー内の検索バー横に配置
- クリックでメガメニュー風のドロップダウンが開く
- 大カテゴリ（13件）をまず表示
- ホバーまたはクリックで中カテゴリ（子）が展開
- 中カテゴリにLevel-2がある場合はさらに展開
- モノタロウのメガメニューを参考にした構造
- 商品件数をカテゴリ名の横に表示（ダミーで可: 例「HPLC (24)」）
- JSで開閉・展開が動作すること

## 画面2: prototype-category-sidebar.html

**検索結果画面のカテゴリ一覧サイドバー**

要件:

- 左サイドバー（幅240-280px） + 右に商品一覧グリッド
- サイドバーに全カテゴリをツリー表示（大 > 中 > 小）
- 現在選択中のカテゴリがハイライト
- 各カテゴリの横に件数表示（ダミー）
- ツリーの開閉がJSで動作
- 商品一覧側にはダミー商品カード4-8件表示
  - 商品カード: 写真プレースホルダー（大きめ）、品名、メーカー、状態ランク、価格
  - 中古機器らしいダミーデータを使用（例: 「島津製作所 HPLC LC-20AD」「エッペンドルフ 遠心分離機 5424R」等）
- サイドバー上部に「カテゴリで絞り込み」の見出し

## 画面3: prototype-breadcrumb.html

**商品詳細画面のパンくずリスト**

要件:

- 商品詳細ページの上部にパンくず表示
- 3階層表示: 「トップ > 分離・クロマトグラフィー > HPLC > （あれば小カテゴリ）」
- 各階層がリンクスタイル
- 商品詳細の内容もそれっぽく表示:
  - 大きな商品写真エリア（プレースホルダー）
  - 品名、メーカー、型番、状態ランク（A/B/C）、価格
  - スペック表（ダミー）
  - 中古品特有の情報: 使用年数、付属品リスト、動作確認状況

## 画面4: prototype-category-always-visible.html

**商品詳細画面のカテゴリ常時表示（新設）**

要件:

- モノタロウの商品詳細ページを参考にしたカテゴリ常時表示エリア
- 商品詳細ページ内（パンくずの下、または商品情報エリア内）に設置
- 「このカテゴリの他の商品」的なセクション
- 同じ中カテゴリ内の他商品をカード形式で3-4件表示
- 親カテゴリ・兄弟カテゴリへのリンク（「分離・クロマトグラフィーの他の機器」等）
- 関連カテゴリのサジェスト（例: HPLCを見ている人に「GC」「質量分析計」も提案）
- 画面3のパンくずと組み合わせた完全な商品詳細ページとして構成
