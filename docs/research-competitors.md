## 1. モノタロウ (monotaro.com)

### メガメニュー/プルダウンの構造

```
┌─────────────────────────────────────────────────────────┐
│ [ロゴ] [カテゴリ▼] [────── 検索ボックス ──────] [カート] │
└─────────────────────────────────────────────────────────┘
         │
         ▼ クリックで展開
┌────────────────────────────────────────────────────────┐
│ 大カテゴリ一覧    │  中カテゴリ一覧（ホバーで展開）      │
│ ─────────────── │  ──────────────────────────────     │
│ ▸ 安全保護用品   │  手袋 / ヘルメット / 安全靴 / ...   │
│ ▸ 物流・保管     │                                      │
│ ▸ オフィス用品   │                                      │
│ ▸ 工具          │                                      │
│ ▸ 測定器具      │                                      │
│   ...           │                                      │
└────────────────────────────────────────────────────────┘
```

**特徴:**
- 検索バーの左に「カテゴリ」プルダウンが配置
- 大カテゴリ（左列）→ 中カテゴリ（右列パネル）の2ペイン構造
- ホバーで中カテゴリが切り替わるフライアウト型
- テキストベースで高密度（アイコン少なめ）
- 各カテゴリに件数や説明は無し（リンクのみ）
- 背景白、ボーダーで区切り、影でフロート表現

**特徴:**

- チェックボックスとラジオボタンでフィルタリング
- テキスト密度が高い（コンパクト）
- 現在選択中のカテゴリは太字 or 色付きで強調

### パンくずリストの実装

```
トップ > カテゴリ > サブカテゴリ > サブサブカテゴリ
```

- ページ上部、ヘッダー直下に配置
- 「>」セパレーター
- 各階層がリンク（最終階層のみテキスト）
- フォントサイズ小（12px程度）
- 色はグレー系、リンクは青

### 商品詳細ページのカテゴリ常時表示

- パンくず下に同カテゴリの関連商品を表示
- 「この商品のカテゴリ」セクション
- 横スクロールのカード列
- 「同じカテゴリの人気商品」のレコメンド

## 2. ZOZO (zozo.jp)

### カテゴリの見せ方

```
ヘッダー:
┌───────────────────────────────────────────────────────┐
│ [ロゴ] [メンズ|レディース|キッズ]  [検索] [お気に入り]  │
│ [カテゴリ] [ブランド] [ランキング] [セール] [コーデ]    │
└───────────────────────────────────────────────────────┘
```

**テキスト/アイコン/写真の使い分け:**
- トップページ: 大きなビジュアルバナー（写真中心）
- カテゴリナビ: テキストリンク（シンプル）
- サブカテゴリ: 小さなサムネイル画像 + テキスト
- ブランド一覧: ロゴ画像 or テキスト

### 余白・フォントサイズの詰め具合

- **フォントサイズ**: 本文 13-14px、見出し 16-20px
- **余白**: コンパクトめ（padding: 8-16px）
- **行間**: 1.5-1.6
- **カード間マージン**: 8-12px
- **全体的に詰め気味**: モノタロウより余白は取るが、一般的なECサイトよりはタイト
- **デザイントークン**: Figmaで管理（色、タイポグラフィ、余白、角丸を体系化）

### 直感的に操作できるUI要素

- **タブ切替**: メンズ/レディース/キッズのタブ
- **ホバーエフェクト**: 商品画像のズーム、色の変化
- **スムーズスクロール**: セクション間の遷移
- **お気に入りボタン**: ハートアイコン（直感的）
- **フィルターバッジ**: 選択中の条件をバッジで表示

### カラーパレット

| 用途 | カラー |
|------|--------|
| プライマリ | 黒 (#000) |
| アクセント | 赤（セール）|
| 背景 | 白 (#FFF) |
| テキスト | 黒 (#333) |
| 補助テキスト | グレー (#999) |
| ボーダー | 薄いグレー (#EEE) |

### カードデザインパターン

```
┌─────────────────┐
│                  │
│  [商品画像]      │  ← 正方形、ホバーでわずかにズーム
│                  │
├─────────────────┤
│ ブランド名       │  ← 小さめ、グレー
│ 商品名           │  ← 14px、黒、1-2行
│ ¥3,990          │  ← 太字、黒
│ ¥5,990 33%OFF   │  ← 取り消し線 + 赤バッジ
└─────────────────┘
```

### デザインの印象・トンマナ

- **ミニマル & モダン**: 装飾少なめ、写真が主役
- **白黒基調**: カラーは写真で、UIはモノトーン
- **余白は程よく詰めている**: 高密度だが窮屈ではない
- **フォント**: ゴシック系（Noto Sans JP系）
- **角丸**: 控えめ（0-4px）
- **影**: ほぼ使わない（フラット）

## 3. プロトタイプへの設計方針（調査まとめ）

### モノタロウから採用するパターン

1. **メガメニュー**: 2ペイン構造（左:大カテゴリ → 右:中カテゴリ展開）
2. **サイドバー**: ツリー形式の開閉 + チェックボックスフィルター
3. **パンくず**: シンプルな「>」区切り階層リンク
4. **カテゴリ常時表示**: 商品詳細ページ内の関連カテゴリセクション
5. **テキスト主体**: アイコン少なめ、リンクテキスト密度高め

### ZOZOから採用するパターン

1. **余白・フォント**: 詰め気味だが整った余白バランス
2. **カードデザイン**: 写真大きめ + コンパクトなテキスト情報
3. **カラー**: モノトーン基調（UIはシンプルに、写真で色を）
4. **ホバーエフェクト**: 控えめだが効果的なインタラクション
5. **角丸・影**: 控えめ（フラット寄り）

### Co-LABO MAKERへの差別化ポイント

1. **写真を大きく**: 中古機器は状態が重要 → 写真で訴求
2. **状態ランク表示**: A/B/C をバッジで明示
3. **スペック情報の視認性**: テーブル形式で整理
4. **信頼感**: 現サイトの青×白の色使いを継承
5. **専門性**: 理化学機器特有のカテゴリ体系
