# Level-2 小カテゴリ提案

作成日: 2026-02-20

## 方針

- 全中カテゴリに小カテゴリを付けるのではなく、商品数が多く細分化に意味がある中カテゴリのみに追加
- 理化学機器業界の一般的な分類（メーカーカタログ、理化学機器総合商社の分類）を参考
- 各小カテゴリ 2〜5 個程度

---
