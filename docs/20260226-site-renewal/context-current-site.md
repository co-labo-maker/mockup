## 5. ご利用ガイドページ (`/guide`)

### ページタイトル

「コラボメーカー中古システム ご利用ガイド」

### 導入文

中古機器・試薬売買サービスの基本的な流れと注意事項を説明するガイド。

### 重要事項ボックス

- **見出し**: 「重要」
- **内容**:
  - 「現状での有姿渡し」（as-is condition delivery）
  - 「ご発注後のキャンセルは承っておりません」

### セクション構成（5大セクション・番号付き）

#### 5-1. 注文方法

##### 1-1. 商品の探し方

3つの検索方法:
- **キーワード検索**: 機器名、メーカー名、型番で検索
- **カテゴリから探す**: 分析機器、実験器具、設備など
- **キャンペーン・特集ページ**: おすすめ商品、人気カテゴリ

掲載内容の説明:
- 商品名、メーカー、型番
- 参考画像
- 用途・仕様
- 価格（保証料・送料別）
- 在庫状況

注記: 一部商品は「在庫確認中・登録準備中」の場合あり

##### 1-2. 見積依頼～注文確定の流れ

4ステップ:

| ステップ | 内容 |
|---|---|
| 1. お問い合わせ・見積依頼 | 商品ページの「見積依頼」ボタンから、希望商品・納品先住所・顧客情報をフォーム送信 |
| 2. お見積書の発行 | 在庫・価格・納期確認後、商品価格（保証料込み）・送料・保証内容をメール送付。電話・メールで詳細確認の場合あり |
| 3. ご発注 | 見積書確認後、メールで発注書を送付。発注は保証条件・ガイド内容への同意を含む |
| 4. 納品・検収 | 保証期間内に機能検証を実施。不具合は保証期間内に報告（返品セクション参照） |

#### 5-2. お支払いについて

##### 2-1. 価格の内訳

| 項目 | 説明 |
|---|---|
| 商品本体の金額 | 保証料込み |
| 送料 | 重量・サイズ・地域・配送条件により個別見積 |
| オプション費用 | 特殊搬入（クレーン、階段搬入）、設置作業など |

##### 2-2. お支払い条件

- **支払方法**: 請求書払い（銀行振込）
- **支払期限**: 月末締め・翌月末払い

#### 5-3. 配送について

##### 3-1. 配送方法
- 配送業者は商品・保管場所により決定
- 配送業者の選択は不可
- 海外・離島・山間部は事前相談が必要

##### 3-2. 送料の考え方

個別見積の基準:
- 商品のサイズ・重量・数量
- 配送先住所
- アクセス条件（エレベーター有無、階段のみ、クレーン必要）
- 保管場所

##### 3-3. 納期の目安
- 注文確定後に確定
- 見積依頼時に希望日を指定可能
- 季節・距離等の要因で希望に沿えない場合あり
- 柔軟なスケジューリングを推奨

##### 3-4. 受け取り時のお願い

到着時の確認事項:
- 破損の有無
- 注文内容との一致

破損・相違があれば直ちに連絡すること。

#### 5-4. 返品について

##### 4-1. 基本方針

「お客様のご都合による返品・交換は原則としてお受けしておりません」

返品不可の例:
- 設置スペースの計算間違い

##### 4-2. 保証・初期不良対応

- **標準保証期間**: 納品日から起算して7日間（ジャンク品・保証除外品を除く）
- **対象不良**:
  - 電源が入らない・起動しない
  - 事前説明と異なる破損
- **重要警告**: 「ご自身で分解・改造・修理を行わないでください」（保証が無効になる）

##### 4-3. 初期不良時の手続き

4ステップ:
1. 保証期間内にメール/フォームで連絡
2. 症状の詳細（発生時期、エラーメッセージ、環境）を提供
3. 要求があれば写真・動画を提出
4. 修理・交換・返品の判断を受領

## 主要な気づき・設計特徴

1. **検討リストは空の状態でしかフェッチできなかった**: 商品追加後のUI構造は不明。ただし、ヘッダーの検討リストアイコンが全ページに存在するため、商品詳細ページから追加する導線があると推測される。

2. **商品事例ページはカテゴリ単位の単一商品紹介**: `/example/hplc` はHPLC全体の紹介ページであり、個別商品の過去取引事例リストではない。`/example` 親ページは存在しない（404）。

3. **カテゴリページのフィルターが充実**: キーワード検索・カテゴリ絞り込み（件数付き）・販売状態フィルターがサイドバーに集約。

4. **価格表示のパターン**:
   - 税込価格表示: ¥XXX,XXX（税込）

6. **保証に関する矛盾**: ホームページのFAQでは「直送スタイルのため保証はございません」とあるが、ガイドページでは「納品日から起算して7日間」の保証が記載されている。ヒーローセクションでも「メンテナンス・稼働保証付き商品多数！」と訴求している。
